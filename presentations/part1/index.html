Проблема: рутина

Что отнимает время

Рутина команды

  • Заполнение отчётов
  • Типовые ответы на запросы
  • Сбор информации из разных систем

Последствия

  • Меньше времени на важные задачи
  • Усталость от однообразия
  • Ошибки из-за невнимательности

До 40% рабочего времени уходит на рутину, которую можно автоматизировать

Рутина в разработке АльфаБанка

Документация

Обновление README, описание API, changelog — до 3 часов в неделю

Code Review

Типовые комментарии, проверка стиля, поиск одинаковых ошибок

Тестирование

Написание unit-тестов для простой логики, моки, фикстуры

В среднем разработчик тратит 6-8 часов в неделю на задачи, которые AI делает за минуты

Цена рутины для команды

Потеря мотивации

  • Разработчики хотят писать код, а не отчёты
  • Скука ведёт к ошибкам
  • Текучка кадров

Замедление Time-to-Market

  • Фичи ждут в очереди
  • Спринты переносятся
  • Бизнес недоволен скоростью

Команда из 10 разработчиков теряет ~60 часов в неделю на рутину — это полтора FTE
